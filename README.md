# Currencies

Implementations for the currency interface from the package compono-kit/money-interfaces

## Usage

Just instantiate the currency named by the iso code (UPPER CASE).
e.g. `new EUR()`

## Requirements

* PHP >= 8.0
* compono-kit/money-interfaces
